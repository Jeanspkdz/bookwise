# 📚 BookWise - Library Management System

## 📌 System Overview
BookWise is a web-based library management system that allows students to browse, borrow, and manage books online. The system includes authentication, an administrative dashboard, and automated email notifications to enhance user experience and system efficiency.

---

## 👥 Actors
- **Guest User** (not authenticated)
- **Authenticated User (Student)**
- **Administrator**
